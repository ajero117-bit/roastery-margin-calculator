# Roastery Margin Calculator

Korean roastery margin calculator built with React and Vite.

## What It Calculates

- Green-bean blend cost per kg
- Roasted coffee cost after roasting loss
- Recommended selling price from target margin
- Actual kg margin when entering your own selling price
- Monthly gross profit, contribution profit, operating profit, and break-even sales
- Variable costs, sales fees, labor, rent, and other fixed costs

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Static Preview

```bash
npm run build
node static-server.mjs
```

Then open `http://127.0.0.1:5173/`.
